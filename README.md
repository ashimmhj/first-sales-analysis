# First Sales Analysis Project

## Overview
This project analyzes sales data to identify revenue trends, best-selling products, and regional performance. The analysis was done using Python, Pandas, and Matplotlib.

## Dataset
- `sales.csv` contains sales records with columns: Date, Product, Region, Price, Quantity
- Revenue is calculated as `Price * Quantity`

## Analysis Performed
- Total revenue calculation
- Best-selling product identification
- Region-wise sales comparison
- Monthly sales trends and visualization

## Key Insights
- **Laptops** generate the highest revenue among all products
- **Asia** contributes the most to overall sales
- Sales show **growth in later months**
- Product pricing has a **significant impact** on total revenue

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## How to Run
1. Download the repository
2. Open `sales_analysis.ipynb` in Jupyter Notebook
3. Make sure `sales.csv` is in the same folder
4. Run all cells to see the analysis and visualizations
